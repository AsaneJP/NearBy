# 簡易仕様書サンプル

### 作者
岡田建人
### アプリ名
NearBy

#### コンセプト
食べに行きたいお店がすぐ見つかる。

#### こだわったポイント
シンプルなデザインですが、料理のお店という事で暖色系を多く使い美味しく見えやすくしました

https://near-by.vercel.app/

## アプリケーション機能

### 機能一覧
- レストラン検索：ホットペッパーグルメサーチAPIによりお店の情報を取得する
- 現在位置の取得：Geolocation APIにより現在位置を取得する

### 画面一覧
- 一覧画面 ：近くのお店の一覧が表示され、絞り込み検索が出来る
- お店情報画面 ：お店の詳細情報が表示される

### 使用しているAPI,SDK,ライブラリなど
- ホットペッパーグルメサーチAPI
- Geolocation API
- cors-anywheren
- Recoil
